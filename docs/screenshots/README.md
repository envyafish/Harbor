# Screenshots / 截图

Put product screenshots in this folder. Filenames below are referenced by the project README.

将产品截图放在此目录。下列文件名已在 README 中引用：

| File | Description |
|------|-------------|
| `home.png` | Home / 首页 |
| `library.png` | Media library / 媒体库 |
| `detail.png` | Title detail / 详情页 |
| `player.png` | Player / 播放器 |
| `settings.png` | Settings / 设置 |

PNG or WebP recommended. Suggested width ≈ 1600–1920px.
